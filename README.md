Batch Inventory System

To run the project after cloning

1. Navigate into the project dir

```
cd batch_inventory
```
2. Create a virtual env

```
python -m venv venv
```
3. Activate the venv

```
venv\Scripts\activate
```

4. Install the dependencies

```
pip install -r requirements.txt
```

5. Run the server

```
fastapi run
```
